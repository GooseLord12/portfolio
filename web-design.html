ActionVault

ActionVault helps collectors organize, track, and showcase their action figure collections. Whether you collect vintage Star Wars figures, Marvel Legends, model kits, or any other line, ActionVault gives you a central place to manage it all! Be it updating figure listings/conditions, sorting through your collection or simply building your own wishlist, these are but some of ActionValut's features that can all be operated within the app, and ActionValut's sleek, yet impactful UI will alow users to navigate these components with ease.

HTML/CSS JavaScript Node.js Ollama API

Future Work: When I revisit this project, I'll add a sort dropdown so collections can be organized by name, value, or date added, implement export/import so collections aren't locked to a single browser, and fully implement a user account and server-based authentication and image handling for efficient functionality.

Uncle Roger's Comic Shop

A multi-page marketing site for a fictional local comic book shop. Features a full light/dark mode toggle that respects user preference and OS settings, a filterable product catalog of 15 comics across 5 categories, and a contact form with topic selection. Built with vanilla HTML, CSS, and JavaScript with no frameworks.

HTML/CSS JavaScript localStorage

Future Work: When I revisit this project, I'll extract the duplicated inline scripts into a single shared script.js file, replace the inline onclick handlers with proper event listeners, wire the contact form to a real backend service and swap the static map placeholder for an embedded interactive map.
